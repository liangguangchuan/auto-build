# auto-build
golang 自动编译工具

Install:
    go install -ldflags "-s -w" github.com/liangguangchuan/auto-build
Usage:
    auto-build golang

welcome start xiaochuan auto-build
 AVAILABLE COMMANDS

    golang      运行应用程序监听当前目录

Use auto-build help [command] for more information about a command.
